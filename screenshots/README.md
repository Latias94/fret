# Screenshots

Curated repository screenshots used by top-level documentation and marketing-style surfaces.

This directory is intentionally small and hand-managed.

- Keep only a few representative checked-in screenshots here.
- Diagnostics output screenshots do **not** belong here; they are written under `target/fret-diag/`
  and related bundle/session directories.
- If a screenshot is only needed for a narrow workstream or one-off investigation, keep it with the
  owning evidence bundle instead of promoting it to this directory.
