# fret-code-view

Fret UI component(s) for rendering source code blocks with optional tree-sitter based syntax highlighting.

## Features

- `syntax-all`: Enables a broad set of common tree-sitter grammars via `fret-syntax/tree-sitter-languages`.
- `syntax-<lang>`: Enables a specific grammar (e.g. `syntax-rust`, `syntax-typescript`).
